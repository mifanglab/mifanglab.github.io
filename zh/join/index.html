# 招募研究生

我们实验室主要围绕心脑血管影像分析、眼科医学影像、医学大模型后训练与 Agent 方向开展研究，聚焦医工交叉的临床真实问题，既有基础算法研究，也有和临床医院合作的转化课题。

实验室长期欢迎具有扎实数学基础，并对计算机视觉和机器学习有兴趣的优秀研究生加入。

# 研究方向

方向 内容
心脑血管影像分析 血管结构、功能与疾病相关影像标志物建模。
眼科医学影像 围绕眼底、OCT/OCTA 等影像开展分析与临床应用研究。
医学大模型后训练与 Agent 面向医学场景的大模型后训练、多模态推理、工具调用和医疗 Agent。

# 希望你是

  • 计划申请南京大学硕士、普通博士或直博项目；2027 届应届本科生如能获得推免资格，欢迎尽早联系。
  • 对医学影像、医学大模型或医工交叉方向有明确兴趣，有较强的代码能力。
  • 做事踏实，愿意长期投入科研，遇到问题能主动沟通。
  • 有论文成果或开源社区贡献。

所在单位 南京大学（苏州校区）生物医学工程学院
系统招生导师 新 PI 张炯教授
